Explicit Wait Best Practices
  • Wait for Visibility: Element is displayed on page
  • Wait for Presence: Element exists in DOM
  • Wait for Clickable: Element is visible AND enabled
  • Wait for Text: Element text equals/contains expected value
  • Wait for Attribute: Element attribute has expected value
  • Wait for Invisibility: Element is not visible (hidden, removed)
  • Set Timeout: Max wait time (10-30 seconds recommended)
  • Poll Interval: Check frequency (default 500ms)
